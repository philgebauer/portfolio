Personal Site

This was a fun website created to display only the essential information - and about page, a short resume, and a contact page. I love the idea of having a personal website with no frills - just the essentials! This site is fully responsive - the simple header collapses when used on smaller devices like a tablet or phone!





Technology Used

This was created using Angular, Angular-router, Express, HTML5, and CSS3. It is hosted and deployed on Heroku and uses Git as version control.
